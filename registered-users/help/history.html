Registered User Viewing History

This is a log of the user's viewing history.

Top navigation

  • Quick actions - Delete and forget a user from the system.

User details

  • E-mail - Email address for the user.
  • Name - The display name for the user.
  • Last Login - The date that this user was last logged into your app.
  • Plan - The type of monetization plan defined for this user.
  • Parental Control - Shows whether parental controls are enabled for this user.
