Videos are added to pages by adding them to playlists and then adding playlists to a page. A playlist is a collection of videos. Before you add videos to pages, you should have videos organized into playlists.

  1. Open the Layout module.
  2. Click on a page name to open the page details.
  3. Click Add Playlist. A list of playlists that have been created will display in a panel.
  4. Select one or more playlists to add to the page by hovering over the playlist and then clicking Add.
    Note:The number of playlists that can be added is based upon the Page Type that was selected. Some Page Types only allow one playlist to be added.
  5. Click X in the panel to close the list of playlists.

Changing playlist layout

Playlist orientation controls how the video images will appear on the page. To change the playlist orientation, follow these steps.

  1. Open the Pages page.
  2. Click on a page name to open the page details.
  3. Click the ellipsis (...) next to the playlist and select an Orientation.
    • Landscape - Landscape images will be displayed on the page
    • Portrait - Portrait images will be displayed on the page
