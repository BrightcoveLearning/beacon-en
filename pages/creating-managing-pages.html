Pages are used to display video content in a Brightcove Beacon application. To create and manage pages, open the Layout module by clicking Layout in the left navigation. A list of pages that have been created will be displayed in the left margin.

The homepage provides the ability to:

  • Create pages
  • Edit pages
  • Duplicate pages
  • Rearrange page order
  • Hide pages
  • Archive and delete pages

Creating pages

  1. Open the Layout module.
  2. In the side navigation panel, click Create Page. The new page panel will open.
  3. Enter a Page Name. The number of characters allowed exceeds what you would practically want for a page title, so title length is not a concern. Press Enter to save the name.
    Note: After you enter the page name, a system generated, internal name is also created and will be displayed below the page name. This internal name cannot be changed.
    Note: If your account is configured with multiple languages, enter a title using the primary language here. Page names for any additional languages can be entered by editing the page title after the page has been created.
  4. Next choose the page layout, which controls how your video content is displayed on the page. For each layout you can read the description and click the layout from the list to see a skeleton preview display of the layout.
  5. At the bottom of the navigation panel, click Next →.
  6. At this point, you can:
    1. See the page is unpublished by default.
    2. Click PUBLISH to publish the page.
    3. Edit page settings, including page title.
    4. Change the page layout (on hover).
    5. Click Add playlist to add video content to your page.
    6. Configure page availability.
  7. The page settings that can be configured will be displayed in the left margin. For more information on adding videos to a page and configuring page availability, see one of the following topics:
    • Adding Videos to Pages
    • Configuring Page Availability
  8. If you wish to return to the list of pages, click the vertical Back to All Pages link. The page that was created will display in the list of pages.

Editing pages

To edit a page, from the Layout module index page in the side navigation panel, click on the page to edit. The page settings will be displayed.

Editing the page title

To edit the page title, click the gear icon next to the page name and enter a new name. You can also edit the title in multiple languages, if they are enabled for your apps.

Editing the page type

To edit the layout, hover over the current layout then click Change. Click on a new layout then click Select.

Duplicating pages

Duplicating pages can be useful when you need to create a new page that is similar to an existing page. To duplicate a page, hover over the page name and click on Duplicate.

Rearranging page order

The order of the pages in the left navigation determines the order of the display in a Beacon application. To change the page order, click on a page and drag it to a new location.

Archiving and deleting pages

Archiving a page will remove it from the application but it will not delete the page. To archive a page, follow these steps:

  1. Click on a page to open the page details.
  2. Click Archive Page at the bottom of the page details.

The total number of archived pages will appear at the bottom of the left margin.

Click Archived Pages to display a list of archived pages.

To permanently delete an archived page, click Delete. Click Restore to unarchive a page and add it back to the page listing.
