Pages are used to display video content in a Brightcove Beacon application. To create and manage pages, open the Layout module by clicking Layout in the primary navigation. A list of pages that have been created are displayed in the left margin.

The homepage provides the ability to:

  • Create pages
  • Edit pages
  • Duplicate pages
  • Rearrange page order
  • Hide pages
  • Archive and delete pages

Creating pages

  1. Open the Layout module.
  2. In the primary navigation, at the bottom of the page list, click + Create Page. The new page panel opens:
  3. Enter a Page Name. The number of characters allowed exceeds what you would practically want for a page title, so title length is not a concern.
    Note: After you enter the page name, a system generated, internal name is created and is displayed below the page name. This internal name cannot be changed. Your entry here is also the page title, which users see in the your apps, but this can be changed.
  4. Next choose the page layout, which controls how your video content is displayed on the page. For each layout you can read the description and click the layout from the list to see a skeleton preview display of the layout.
    For complete details, including screenshots of how the individual page layout appears in a Brightcove Beacon web app, see the Using Page Layouts document.
  5. At the bottom of the navigation panel, click Create.
  6. At this point, you can:
    1. See the page is unpublished by default.
    2. Click PUBLISH to publish the page.
    3. Edit page settings, including page title.
    4. Change the page layout (on hover).
    5. Click Add playlist to add video content to your page.
    6. Configure page availability.
  7. The page settings that can be configured are displayed in the left margin of the Page Settings modal. For more information on adding videos to a page and configuring page availability see the following documents:
    • Managing Playlists on Pages
    • Configuring Page Availability
  8. If you wish to return to the list of pages, click the vertical Back to All Pages link. The page that was created is displayed in the list of pages.

Editing pages

To edit a page, from the Layout module index page, click on the page to edit. The page settings are displayed.

Editing the page title

To edit the page title, click next to the page title and enter a new title. You can also edit the title in multiple languages, if they are enabled for your apps.

Editing the page type

To edit the layout, hover over the current layout then click Change. Click on a new layout then click Select.

You may see a message stating that a selected page layout may not support the number of playlists you had selected for the previously used layout.

Duplicating pages

Duplicating pages can be useful when you need to create a new page that is similar to an existing page. To duplicate a page, hover over the page name and click .

Rearranging page order

The order of the pages in the left navigation determines the order of the display in a Beacon application. To change the page order, click on a page and drag it to a new location.

Archiving, restoring and deleting pages

Archiving a page removes it from the application but it does not delete the page. To archive a page, follow these steps:

  1. Hover over a page then click next to the page name. This opens the Page Settings dialog.
  2. On the bottom-left of the page settings dialog, click Archive page.
  3. Note that the page is immediately archived and you are returned to the index page of the Layout module with the archived page no longer in the page list.

  3. Click to restore the archived page, or click the to permanently delete the page.
  4. Confirm that you are returned to the index page of the Layout module where you see the page list. If you restored a page it will appear in the page list, but in an unpublished state.
