Pages are used to display video content in a Brightcove Beacon application. To create a page, follow these steps:

  1. Open the Pages page.
  2. In the side navigation panel, click Create Page. The new page panel will open.
  3. Enter a Page Name using the following guidelines.
    • Name is limited to 100 characters
    • Illegal character: /
  4. Press enter to save.
  5. Click on a Page Type to select it. The Page Type controls the layout of the page and how the content is displayed on the page. Selecting a page type will update the page preview on the right.
  6. Click Next.
  7. Click Add Playlist. A list of playlists that have been created will display.
  8. Select one or more playlists to add to the page by hovering over the playlist and then clicking Add. As playlists are added, they will appear on the page preview.
  9. Click X to close the list of playlists.
  10. Click Back to All Pages to return to the list of pages.

The page that was created will display in the list of pages. To edit any of the page details, click on the page name.
