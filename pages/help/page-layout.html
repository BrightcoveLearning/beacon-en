{{ page.description }}
  1. Choose one of the following layouts:
    • Playlist Grid
    • Hero (Autoplay Channel)
    • Hero (Featured Playlist)
    • Hero (Image)
    • Carousels
  2. Note that when a page layout is selected, the page preview will update to reflect the current selection.
  3. After making your selection, click the Select button that is positioned under the container types.

You can also click the gear icon beside the page name to edit the page title and availability.
