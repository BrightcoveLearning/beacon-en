Publish/Unpublish

Publish/Unpublish the page using the button at the top of navigation panel.

Edit page details

Click to edit the page details including:

  • Change page title
  • Control availability
  • Archive page

Change layout

Hover over the current layout, then click Change to choose a new layout.

Add playlist(s)

Click Add Playlist to control the playlist(s) that are displayed on the page.

Alter availability

The Availability settings are used to restrict page availability based upon:

  • Date
  • Device Type
  • Location

To change availability, either click to the right of the Availability label, or the setting you wish to change.
