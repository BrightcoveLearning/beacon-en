Add Videos
  1. Click Add Videos.
  2. Select the videos you want (you can also search for videos).
  3. Click Add Videos.
  4. Click Save to save the changes to the channel.
  • Learn more
Import Playlist
  1. Click Import Playlist.
  2. Select the playlist you want and click the Importbutton beside it.
  3. Click Save to save the changes to the channel.
      • Learn more
Add Overlays
  1. Click Overlays.
  2. Select the playlist you want and click the Importbutton beside it.
  3. Click Save to save the changes to the channel.
  • Learn more

Reorder Videos

Note this applies only if you have added videos and/or live events to the channel.


  1. Click the video asset you wish to move.
  2. Drag it to the new location and drop it.
  • Learn more
Quick Edit

Note this applies only to live events in the program.


  1. Select a live event.
  2. Click Quick Edit.
  3. Adjust the duration of the event.
  4. Click Save.
  • Learn more
Activate Channel

  1. Click Activate.
  2. The channel will play automatically at the start date/time, but you can start it immediately by clicking the Play Now button.

While the channel is playing
View Analytics
  1. View the currently playing video and analytics.
  2. The analytics graphs are useful for seeing problems, especially with live feeds not connecting (dips in input data).
  • Learn more
View in Brightcove Live
  1. Click View in Brightcove Live.
  2. In Brightcove Live, you will see additional options such as getting the publishing code for the event.
  • Learn more
Show Playing Now
  1. Click Show Playing Now to highlight the currently playing video.

This option is only available if the currently playing video is visible in the video list.

  • Learn more
Edit Program
  1. Click Edit Program.
  2. Add, delete, or re-order videos.
  3. Click Save.

Changes will apply to future playing items.
