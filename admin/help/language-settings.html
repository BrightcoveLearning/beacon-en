Language Settings Help

Define the languages associated with your video metadata.

If you delete a language that is in use, all metadata associated with the language will be hidden. To restore the associated metadata, add the language again.

Primary language

The Primary Language is initially displayed when you edit video properties in the Media module. Existing metadata will be associated with the Primary Language.

Edit primary language

  1. Click on the primary language displayed to change it.
  2. Start typing your language to see a list of values.
  3. Select a language.
