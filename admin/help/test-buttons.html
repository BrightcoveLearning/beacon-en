Edit Channel Details
  1. Click .
  2. Edit the details you want to change and click Update Channel.
  3. Delete - Select one or more ad configurations, and click Delete to remove them from your account
  4. Duplicate - Select an ad configuration, and click Duplicate to create an ad configuration copy; you can cancel, or edit and save the new configuration
  5. Enter a name (English only, no spaces; these special characters are allowed: _ . : / = + -).
  6. If you toggle Loop Cloud Playout on, you may also enter an optional End Date/Time.

    When you turn on Loop Playout Content, you may also enter an optional End Date/Time.
  7. Toggle Enable Server-Side Ad Insertion (SSAI) on to enable SSAI - it will also remove the Live Input option.

    For server-side ads, turn on Enable Server-Side Ad Insertion (SSAI) - it will also remove the Live Input option.
  8. Click Add Videos.
  9. You can drag live events to reorder the list just as you can videos. You can also use the Quick Edit button to modify the duration of a live event.
