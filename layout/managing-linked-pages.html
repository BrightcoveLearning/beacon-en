Introduction

If you have a page with a carousel that contains a playlist of more than 25 videos you can create a linked page that will automatically be shown in lieu of the very wide carousel. Normally you would want the large number of videos display in a format, like a grid, where the viewer doesn't have to scroll and scroll to see all the videos.

Note that a linked page can only be defined for a page that displays videos in a carousel. The single exception to this statement is a linked page cannot be created when the Hero (Autoplay Channel) is the layout selected for the page from which you wish to link.

Creating a linked page

The following are the steps required to create a linked page:

  1. In the Layout module navigate to the properties of a page that contains a carousel.
  2. To the right of the playlist whose contents you wish to display in a linked page, click the elipses (...).
  3. In the playlist settings dialog which opens, click More layout options.
  4. In the playlist layout dialog which opens, to the right of Linked page, click Add.
  5. Choose the layout for the linked page.
  6. Click Save.
  7. Click Publish and confirm your action in the dialog that appears.
  8. Return to your page's properties.
  9. You will be returned to the page properties page. Check to be sure you can see the Linked page indicator (highlighted).
For any content-based page layout that has content displayed in a carousel (except the Auto Play layout), open the "options menu" for the playlist that you'd like to use for the attached linked page and click "more layout options" At the bottom of the "more layout options modal", there should be a "Linked page" control with a button to add (or remove) a linked page. Click the add button to attach a linked page to this playlist.
