Steps

  1. Open the Pages page.
  2. In the side navigation panel, click Create Page.
  3. The next dialog has you choose the general page group. In this case, choose Customize with your content.
  4. Enter a Page Name using the following guidelines.
    • Name is limited to 100 characters
    • Illegal character: /
  5. Select a Page Type by clicking on it. The Page Type controls the layout of the page and how the content is displayed on the page. In this case, choose Tabs.
  6. Click Create.
  7. Click Add Playlist. A list of playlists that have been created will display.
  8. Select one or more playlists to add to the page by hovering over the playlist and then clicking Add. The number of playlists that can be added is based upon the Page Type that was selected. Some Page Types only allow one playlist to be added. As playlists are added, they will appear on the page preview.
  9. Click X to close the list of playlists.

You have now created a page and added your video content to it.
