Creating a Smart Playlist

In this topic you will learn how to create a smart playlist.

Steps

  1. Open the Media page.
  2. In the side navigation panel, click the Create Playlist link.
  3. Set the Playlist type to Smart.
  4. Set the Playlist Name to Birds or another appropriate name.
  5. Check Show Playlist in Sidebar to have the playlist appear in the sidebar.
  6. Set the Tags field to contains one or more and enter birds for the tag (or whatever the appropriate tag may be).
  7. Click Save.
  8. Click the playlist name just created in the side navigation panel and confirm the correct videos have been added to the playlist.
