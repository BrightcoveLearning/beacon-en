How To: Create a Smart Playlist

Steps

  1. Open the Media module.
  2. In the panel navigation, click the Create playlist link.
  3. Set the Playlist type to Smart.
  4. Enter a Playlist Name.
  5. Check Show Playlist in Sidebar to have the playlist appear in the sidebar.
  6. Set the Tags field to contains one or more and enter the appropriate tag(s).
  7. Click Save.
  8. Click the playlist name just created in the panel navigation panel and confirm the correct videos have been added to the playlist.
