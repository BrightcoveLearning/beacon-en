Introduction

If you have a page with a carousel that contains a playlist of more than 6 or 10 (depending on platform) a View All link appears at the end of the carousel.

There is a default view all page, which you can alter. Also, you can create a secondary page that automatically shows in lieu of the very wide carousel. Normally you would want the large number of videos display in a format, like a grid, where the viewer doesn't have to scroll excessively to see all the videos.

Note that a secondary page can only be defined for a page that displays videos in a carousel. The single exception to this statement is a secondary page cannot be created when the Hero (Autoplay Channel) is the layout selected for the page from which you wish to link.

Note: For clarity, in the steps below the term main page refer to the page where you choose the playlist which displays the Secondary page. On the main page you see a Secondary page indicator (highlighted) associated with the playlist added to the main page.

Creating a secondary page

The following are the steps required to create a secondary page:

  1. In the Layout module navigate to the properties of a page that contains a carousel.
  2. To the right of the playlist whose contents you wish to display in a secondary page, click the ellipses (...).
  3. In the playlist settings dialog which opens, click More layout options.
  4. In the playlist layout dialog which opens, to the right of Secondary page, click Add.
  5. Choose the layout for the secondary page.
  6. Click Create.
  7. Confirm you are now viewing the properties for the secondary page.
  8. Click Publish and confirm your action in the dialog that appears.
  9. Return to your main page's properties.
  10. Check to be sure you can see the Secondary page indicator (highlighted).

Edit a secondary page

To edit an existing secondary page, click on the Secondary page indicator associated with the playlist added to a main page.

You then see the secondary page's details.

Delete a secondary page

To delete an existing secondary page, click on the Secondary page indicator associated with the playlist added to the main page.

You then see the secondary page's details, where you can click Delete.

Access secondary pages from the page list

So far in the document you have accessed details about your secondary page by clicking on the Secondary page indicator associated with the playlist added to the main page. You can also access secondary pages from the index page of the Layout module. To do so, follow these steps:

  1. From the index page of the Layout module, scroll to the bottom of the page list until you see the secondary pages text, which is followed by the number of secondary pages in your experience.
  2. Click secondary pages.
  3. Check to be sure you see a list of all the secondary pages you have created.
  4. Click one one of the links and you see the secondary page's details.
  5. From here you can edit, delete or publish your secondary page. Note that you cannot set availability on secondary pages.
