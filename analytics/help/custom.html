Custom Reports

Create custom analytics reports that show selected data and metrics for a period of time.

Report time zone
  • Data in custom reports is based on the account's reporting time zone; to configure your time zone:
    1. In Video Cloud Studio, expand the Admin dropdown.
    2. Select Account Settings.
    3. Set your Current Time Zone.
    4. Click Save.
    Changing the reporting time zone only affects data going forward, and changes are not applied retroactively.

Report data
  • Include Accounts - Include data from the current Video Cloud account or select multiple accounts
  • Report Type
    • One-time - Report is run once, immediately
      • Date Range - For one-time reports, specify the time period for analytics data
    • Recurring - Report can be scheduled to run on a daily, weekly or monthly basis
      • Frequency
        • Daily - Report covers an entire day
        • Weekly - Report covers an entire week, defined as Monday through Sunday
        • Monthly - Reports covers an entire month
      Recurring reports are executed after analytics reconciliation, usually 3 to 5 days after the end of a reporting period.
  • Select Show title and description to display the video title and description on the player
  • Play Button Shape - Controls the shape of the play button:
    • Circle
    • Square
    • Rectangle
  • Play Button Position - Controls the location of the play button within the player:
    • Center
    • Top Left
    • Top Right
  • Color Theme - Controls the colors used in the player including the play/pause buttons, elapsed time, volume, captions, full screen and sharing icons.
  • Logo Overlay - Add an image on top of the player:
    • On - Upload an image and position it:
      • Bottom Right
      • Bottom Left
      • Top Right
    • Off - No image
  • Click Save to save your changes
